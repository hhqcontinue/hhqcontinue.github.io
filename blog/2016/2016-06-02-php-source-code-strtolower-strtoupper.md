将字符串转换成大写字符。

## 源码解读

两个函数的核心操作都差不多，讲一下strtolower，另一个是类似的。
php_strtolower函数的核心代码如下：

    c = (unsigned char *)s;
    e = c+len;

    // 遍历s，逐个变为小写
    while (c < e) {
    　　*c = tolower(*c);
    　　c++;
    }
    return s;
 

这个函数就是遍历整个字符串，逐个转成小写字符。这也是一个经典的指针操作。
